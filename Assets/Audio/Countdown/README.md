# カウントダウン音声ファイル配置ガイド

## フォルダ構造

```
Assets/Audio/Countdown/
├── StartCountdown/     # ゲーム開始前のカウントダウン音声
│   ├── 3.wav          # "3"のボイス
│   ├── 2.wav          # "2"のボイス
│   ├── 1.wav          # "1"のボイス
│   └── start.wav      # "ゲーム開始！"のボイス
└── EndCountdown/       # ゲーム終了前のカウントダウン音声
    ├── 10.wav         # "10"のボイス
    ├── 9.wav          # "9"のボイス
    ├── 8.wav          # "8"のボイス
    ├── 7.wav          # "7"のボイス
    ├── 6.wav          # "6"のボイス
    ├── 5.wav          # "5"のボイス
    ├── 4.wav          # "4"のボイス
    ├── 3.wav          # "3"のボイス
    ├── 2.wav          # "2"のボイス
    ├── 1.wav          # "1"のボイス
    └── end.wav        # "ゲーム終了！"のボイス
```

## 使用方法

1. 上記のフォルダ構造に従って音声ファイルを配置してください
2. Unityエディタで `CountdownManager` コンポーネントのInspectorから音声ファイルを設定してください
3. `BGMManager` のカウントダウン用 `AudioSource` を設定してください

## 注意事項

- ファイル形式は .wav または .mp3 を推奨します
- ファイル名は上記の通りに設定してください（大文字小文字を区別します）
- 音声の長さは1秒程度を推奨します
