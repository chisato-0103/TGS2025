# Shoot of Love ~ハーレムゴリラ編~

東京ゲームショー2025 展示作品

## 概要

**Shoot of Love ~ハーレムゴリラ編~** は、ゴリラの求愛行動をテーマにした豪快なシューティングゲームです。
カースト最下位のゴリラ（主人公）が、うんこを投げてメスゴリラを求愛し、ハーレムを築いて下剋上を目指します。

ゴリラ型の自作コントローラーを使用した体感型ゲームで、腕を動かしてうんこを投げる独特な操作感が特徴です。

## 世界観・コンセプト

### 🦍 ゴリラの求愛って面白いよね

ゴリラの求愛はドラミング以外にも求愛方法が存在します。それは**うんちを投げること**です。
うんち投げは求愛以外でも威嚇の意味ですることがあります。

### 🌴 世界観

- 舞台はジャングル（マフィア街）
- メスゴリラ（ハーレム）の数でカーストが変わる社会
- オスゴリラは個々の得意分野でメスゴリラを求愛し、ハーレムを作る
- カースト最下位レベルのゴリラ（主人公）がうんこで求愛する方法を発見
- 最弱からゴリラ界トップを目指す下剋上ストーリー

## ゲームの特徴

- **ジャンル**: シューティングゲーム（豪快・バカゲー）
- **プレイ人数**: 1人
- **制限時間**: 60秒
- **目的**: うんこを標的に向かって投げて、メスゴリラの注目を集めハーレムを築く
- **リザルト**: 結果に応じてリザルト画面が華やかに変化

## 操作方法

### 🎮 ゴリラ型コントローラー

本ゲームはキーボードやマウスを使用せず、**自作のゴリラ型コントローラー**で操作します。

#### ゴリラの腕（上下の狙い）
- 右腕を後ろに倒すことができる
- 倒した腕はキックバネの力で元の状態に戻る
- 腕を倒している時間で上下の狙いを調整

#### 回転台座（左右の狙い）
- ゴリラが台座の上に乗る
- 台座を回転させる
- 左右方向の回転で横方向の狙いをつける

### 🔧 センサー

**M5stick CPlus** を2台使用：
- **ゴリラの腕**: 腕が後ろに倒されているかを検出
- **回転台座**: 台座の回転角度を検出

## 開発環境

### 必要なもの

- Unity 6000.1.3f1 以上
- Visual Studio または Visual Studio Code
- M5stick CPlus × 2台（実機プレイ用）

### プロジェクトのセットアップ

1. リポジトリをクローン
```bash
git clone https://github.com/chisato-0103/TGS2025.git
cd TGS2025
```

2. Unity Hubでプロジェクトを開く
   - Unity ver.6000.1.3f1 を使用してプロジェクトを開く

3. シーンを開く
   - `Assets/Scenes` フォルダ内のシーンを開く

4. プロジェクトを実行
   - Unity エディタの再生ボタンでゲームを実行

## プロジェクト構成

```
Assets/
├── Scenes/          # ゲームシーン
├── Script/          # C#スクリプト
│   ├── GameScene/   # ゲーム本編のスクリプト
│   ├── RankingScript/  # ランキング機能
│   └── ResultScript/   # リザルト画面
├── Prefabs/         # プレハブ
├── image/           # 画像素材
└── Settings/        # プロジェクト設定
```
